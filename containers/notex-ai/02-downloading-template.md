# Step 2: Downloading Ubuntu Template

**Date:** 2026-02-18

## Process
I didn't have an Ubuntu template yet, so I needed to download one.

## Steps Taken
1. In Proxmox web UI, went to: **pve node** → **local storage** → **CT Templates**
2. Clicked **Templates** button
3. Searched for "ubuntu"
4. Selected **ubuntu-24.04-standard** 
5. Clicked **Download**

## Status
⏳ Downloading (takes a few minutes)
